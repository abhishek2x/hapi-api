# hapi-api

***An introduction to the hapi Node.js Framework***


## Getting Started

#### Create a new directory for the project
> $ mkdir hapi-api && cd hapi-api

#### Initialise a new Node project
> $ npm init -y

#### Install hapi.js
> $ npm init hapi

#### Create a new server file
> $ touch index.js

#### Install nodemon
> $ npm i nodemon -g

#### Run server with nodemon
> $ nodemon index.js
